# 📐 KPIs y Métricas

Esta página define qué vamos a medir, por qué y cómo sabremos si la herramienta está funcionando. Alineado con el Dashboard de Accesibilidad - Moodle y el esquema de backend confirmado por Leandro.

## Objetivo de medición

Entender si la herramienta de accesibilidad reduce las barreras de navegación y comprensión dentro del aula virtual, y si los estudiantes logran completar su recorrido de forma más autónoma.
* * *

## KPIs principales

### Adopción general

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Usuarios activos | COUNT DISTINCT usuarios con al menos una sesión en el período | Diaria | Sessions.moodle\_user\_sub |
| Sesiones totales | COUNT session\_id | Diaria | Sessions |
| Eventos totales | COUNT event\_id | Diaria | EventLog |
| Duración promedio de sesión | AVG(ended\_at - started\_at) WHERE ended\_at IS NOT NULL | Por sesión | Sessions |
| Tareas completadas | COUNT eventos WHERE event\_type = 'task\_completed' | Diaria | EventLog |
| Usuarios nuevos vs recurrentes | Nuevos: created\_at dentro del período / Recurrentes: el resto | Por período | Users.created\_at |
| Variación vs período anterior | % de cambio de cada KPI respecto al mes anterior | Por período | Gold — lógica de ventana temporal |

### Accesibilidad

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Ajustes más usados | COUNT eventos accessibility\_changed GROUP BY payload.adjustment\_type | Por sesión | EventLog.payload |
| Contraste preferido | Distribución de contrast\_mode actual por usuario (Alto / Normal) | Por período | Users.accessibility\_settings.contrast\_mode |
| Detalle de ajuste | Qué parámetro se cambió, de qué valor a cuál | Por evento | EventLog.payload (old\_value / new\_value) |

### IA cognitiva

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Solicitudes por tipo | COUNT ai\_cognitive\_request GROUP BY payload.request\_type (summary / simplify / key\_concepts) | Diaria | EventLog.payload |
| Evolución diaria por tipo | COUNT GROUP BY DATE(timestamp) + payload.request\_type | Diaria | EventLog.payload |
| Tasa de cache hit | % de solicitudes servidas desde AiCache vs generadas nuevas | Por período | EventLog.payload.cache\_hit |

### Texto a voz (TTS)

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Acciones TTS | COUNT tts\_interaction GROUP BY payload.action (play / pause / stop) | Por sesión | EventLog.payload |
| Evolución diaria TTS | COUNT GROUP BY DATE(timestamp) | Diaria | EventLog.timestamp |

### Perfiles rápidos (presets)

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Presets más usados | COUNT preset\_selected GROUP BY payload.preset\_name | Por período | EventLog.payload |
| Distribución por preset | % de cada preset sobre el total de activaciones | Por período | EventLog.payload |
| Evolución diaria por preset | COUNT GROUP BY DATE(timestamp) + payload.preset\_name | Diaria | EventLog.payload |

### Uso por curso y ciudad

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Sesiones por curso | COUNT GROUP BY moodle\_course\_id | Por período | EventLog.payload + MoodleCourseCache |
| Uso por ciudad | COUNT sesiones GROUP BY city | Por período | user\_enrichment.city |
| Curso con mayor adopción | Curso con más usuarios activos únicos | Por período | EventLog + MoodleCourseCache |

### Navegación y progreso

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Funciones más usadas | COUNT navigation\_event GROUP BY payload.destination | Por sesión | EventLog.payload |
| Modo concentración | % sesiones con modo activado vs no activado | Por período | EventLog (concentration\_mode\_toggle) ⚠️ pendiente backend |
| Progreso promedio por curso | AVG(status → %) GROUP BY moodle\_course\_id | Por período | UserProgress.status |
| Actividades pendientes | COUNT WHERE status = 'PENDING' | Por usuario | UserProgress |
| Actividades en progreso | COUNT WHERE status = 'IN\_PROGRESS' | Por usuario | UserProgress |
| Retención | Última interacción por usuario y recurso | Por usuario | UserProgress.last\_interaction\_timestamp |

### Calidad del producto

| KPI | Definición | Frecuencia | Fuente |
| ---| ---| ---| --- |
| Tasa de éxito | % eventos WHERE resultado = 'SUCCESS' | Por sprint | EventLog.resultado |
| Errores registrados | COUNT WHERE resultado = 'FAILED' | Por sprint | EventLog.resultado |

## Preguntas que queremos responder con datos

| Pregunta | Cálculo | Fuente |
| ---| ---| --- |
| ¿Los estudiantes usan las herramientas o las ignoran? | COUNT(event\_type) por usuario, excluyendo session\_start | EventLog |
| ¿Qué herramienta genera más valor? | COUNT(event\_type) GROUP BY event\_type ORDER BY count DESC | EventLog |
| ¿Los usuarios que usan TTS o resúmenes avanzan más? | Cruzar tts\_interaction + ai\_cognitive\_request con UserProgress.status = 'COMPLETED' | EventLog + UserProgress |
| ¿En qué punto del flujo se abandonan las sesiones? | Último event\_type por session\_id antes del cierre | EventLog + Sessions |
| ¿Cuántos usuarios configuran sus preferencias visuales? | COUNT DISTINCT moodle\_user\_sub WHERE event\_type = 'accessibility\_changed' | EventLog |
| ¿Qué preset es el más adoptado por perfil de usuario? | COUNT preset\_selected GROUP BY payload.preset\_name + user\_enrichment.accessibility\_profile | EventLog + user\_enrichment |
| ¿Qué ciudad usa más la herramienta? | COUNT sesiones GROUP BY city | Sessions + user\_enrichment |
| ¿Qué tipo de solicitud de IA es la más demandada? | COUNT ai\_cognitive\_request GROUP BY payload.request\_type | EventLog.payload |

## Event types del tracking — referencia rápida

| event\_type | Qué registra | Campos clave en payload |
| ---| ---| --- |
| accessibility\_changed | Cambio de ajuste visual | adjustment\_type, old\_value, new\_value |
| ai\_cognitive\_request | Solicitud de IA cognitiva | request\_type, activity\_id, moodle\_course\_id |
| tts\_interaction | Interacción con texto a voz | action, activity\_id, moodle\_course\_id |
| preset\_selected | Activación de perfil rápido | preset\_name, preset\_id |
| navigation\_event | Navegación entre secciones | destination, origin |
| task\_completed | Tarea completada | activity\_id, moodle\_course\_id |
| session\_start | Inicio de sesión | — |
| concentration\_mode\_toggle | Modo concentración ⚠️ pendiente | state (on / off) |
